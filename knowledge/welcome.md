# Welcome to the public-agent knowledge base

This is a sample knowledge file. Operators replace the contents of the
`knowledge/` directory with content they want the public-agent to be able
to quote from.

Rules:

- Files must be Markdown with a `.md` extension.
- Filenames must be lowercase and dash-separated (for example, `welcome.md`,
  `about.md`, `pricing-faq.md`).
- Files are capped at 64 KB each.
- The agent cannot modify files — it can only search and read.
